
## 样本还是随机变量? 期望和分布的严格Notation

期望角标和括号里的 \\(X\\) 应该大写还是小写？这个问题看似很简单，但实际并没有那么显然。ML 里的符号有很多工程化简写，再加上不同学科的符号习惯并不完全一致，导致同一个符号在不同语境下可能承担不同角色。

符号是定义的体现。所以这一节先从严格的数学视角出发，区分随机变量、取值、样本和分布；再回到 ML 论文中常见的简写，说明这些简写到底省略了什么。

### 1. 分布符号的严格写法

在严格概率论中，“服从某个分布”的对象应该是随机变量，而不是一个具体取值。通常我们用大写字母表示随机变量，用小写字母表示随机变量可能取到的值。

例如，设随机变量 \\(X\\) 的取值空间为 \\(\mathcal{X}\\)，它的概率密度函数或概率质量函数为 \\(p(x)\\)。严格地说，我们可以写：

\\[
X \sim p
\\]

或者在连续分布语境下，为了强调密度函数在点 \\(x\\) 处的值，也常写成：

\\[
X \sim p(x)
\\]


这里各个符号的含义是：

- \\(X\\)：随机变量，也就是从概率空间到取值空间 \\(\mathcal{X}\\) 的映射。
- \\(x\\)：随机变量 \\(X\\) 的一个可能取值，是取值空间里的点。
- \\(p(x)\\)：随机变量 \\(X\\) 的密度函数或质量函数在 \\(x\\) 处的值。
- \\(x_i\\)：一个已经观测到的样本，通常可以理解为某个随机变量 \\(X_i\\) 在一次实验结果 \\(\omega_i\\) 上的取值，即 \\(x_i = X_i(\omega_i)\\)。

所以，从严格意义上说，下面这种写法并不准确：

\\[
x \sim p(x)
\\]

因为 \\(x\\) 是一个取值，不是随机变量。一个点本身不会“服从分布”，只有随机变量才谈得上“服从分布”。

不过在 ML 论文和代码伪代码中，\\(x \sim p(x)\\) 非常常见。它通常不是严格的概率论表述，而是一种工程化简写，真实含义是：

> 从分布 \\(p\\) 中采样一个值，并把这个值记作 \\(x\\)。

因此，在直觉解释或算法伪代码中，写 \\(x \sim p(x)\\) 通常可以接受；但在严谨推导中，最好先区分随机变量和取值，例如写成 \\(X \sim p(x)\\)，再用 \\(x\\) 表示积分变量或具体取值。

### 2. 期望符号的严格写法

期望的括号里放的是随机变量，或者由随机变量构造出来的新随机变量。例如：

\\[
\mathbb{E}[X]
\\]

表示随机变量 \\(X\\) 的期望。如果我们关心的是 \\(X\\) 经过函数 \\(f\\) 之后的结果，则写成：

\\[
\mathbb{E}[f(X)]
\\]

这里 \\(f(X)\\) 仍然是一个随机变量，因为 \\(X\\) 是随机的，\\(f(X)\\) 的值也会随实验结果变化。

如果 \\(X\\) 有密度函数 \\(p(x)\\)，那么期望可以展开成积分形式：

\\[
\mathbb{E}[f(X)] = \int_{\mathcal{X}} f(x)p(x)dx
\\]

这里需要特别注意 \\(X\\) 和 \\(x\\) 的区别：

- \\(X\\)：随机变量。
- \\(x\\)：积分变量，也叫 dummy variable，只是积分表达式里的占位符。
- \\(f(X)\\)：随机变量 \\(X\\) 经过函数 \\(f\\) 得到的新随机变量。
- \\(f(x)\\)：函数 \\(f\\) 在具体取值 \\(x\\) 上的函数值。

所以，\\(\mathbb{E}[f(X)]\\) 和 \\(\int f(x)p(x)dx\\) 是同一个对象的两种表达方式：前者强调随机变量，后者强调对取值空间做积分。

在 ML 中，我们常把上式简写成：

\\[
\mathbb{E}_{x \sim p(x)}[f(x)]
\\]

这是一种非常常见的写法，但它混合了两层含义：

1. 角标里的 \\(x \sim p(x)\\) 像是在说“从分布中采样”。
2. 括号里的 \\(f(x)\\) 又把 \\(x\\) 当成积分变量或样本值。

如果写成积分，它真正对应的是：

\\[
\mathbb{E}_{x \sim p(x)}[f(x)]=\int f(x)p(x)dx
\\]

因此，使用带角标的期望时，一个比较稳妥的规则是：角标里如果写了 \\(x\\)，括号里也使用同一个小写 \\(x\\)，把它理解为积分变量。例如：

\\[
\mathbb{E}_{x \sim p(x)}[f(x)]
\\]

而不要写成：

\\[
\mathbb{E}_{x \sim p(x)}[f(X)]
\\]

后一种写法容易把随机变量 \\(X\\) 和积分变量 \\(x\\) 混在一起。

如果希望既严谨又不啰嗦，可以先写清楚随机变量的分布：

\\[
X \sim p(x)
\\]

然后写：

\\[
\mathbb{E}[f(X)] = \int_{\mathcal{X}} f(x)p(x)dx
\\]

在后续推导中，再根据上下文简写为：

\\[
\mathbb{E}_{x \sim p(x)}[f(x)]
\\]

这样读者既能看到严格对象 \\(X\\)，也能看懂 ML 论文中常见的简写形式。

### 3. 期望角标的本质：对测度积分

这时我们会遇到一个问题：为什么期望的角标里会出现 \\(x \sim p(x)\\) 这种看起来不太严谨的写法？一个 dummy variable 怎么会“来自一个分布”？这似乎和前面“只有随机变量才能服从分布”的说法矛盾。

要解释这个问题，需要从测度论视角重新看一遍概率、随机变量、分布和期望之间的关系。

#### 3.1 概率是什么：概率是一个测度

在测度论里，概率是一种特殊的测度。一个概率空间写作：

\\[
(\Omega, \mathcal{F}, \mathbb{P})
\\]

其中：

- \\(\Omega\\)：样本空间，表示所有可能的实验结果。
- \\(\mathcal{F}\\)：事件集合，是样本空间上的一个 \\(\sigma\\)-代数，表示我们可以讨论概率的可测集合。
- \\(\mathbb{P}\\)：概率测度，给每个事件分配一个 \\([0,1]\\) 之间的数。

也就是说，概率测度是一个函数：

\\[
\mathbb{P}: \mathcal{F} \to [0,1]
\\]

它至少满足三个核心性质：

- 非负性：对任意事件 \\(A \in \mathcal{F}\\)，都有 \\(\mathbb{P}(A) \geq 0\\)。
- 规范化：整个样本空间的概率为 \\(\mathbb{P}(\Omega)=1\\),空集合 \\(\mathbb{P}( \varnothing )=0\\).
- 可列可加性：如果 \\(A_1,A_2,\dots\\) 两两不相交，则

\\[
\mathbb{P}\left(\bigcup_{i=1}^{\infty} A_i\right)=\sum_{i=1}^{\infty}\mathbb{P}(A_i)
\\]

所以，概率的测度论本质是：它给事件集合赋值，并且这个赋值方式满足测度的基本规则。

#### 3.2 随机变量是什么：从样本空间到取值空间的映射

随机变量 \\(X\\) 并不是普通意义上的“变量”，本质上是一个函数，是从样本空间到取值空间的映射：

\\[
X: \Omega \to \mathcal{X}
\\]

也就是说，\\(X\\) 会把每个实验结果 \\(\omega \in \Omega\\) 映射成一个取值 \\(x \in \mathcal{X}\\)。

例如，掷一次骰子时，\\(\Omega\\) 可以是所有可能的实验结果，\\(\mathcal{X}\\) 可以是 \\(\{1,2,3,4,5,6\}\\)。随机变量 \\(X\\) 的作用，就是把一次实验结果映射成我们关心的点数。

再比如高斯随机变量，\\(\Omega\\) 往往是一个抽象概率空间，而 \\(\mathcal{X}=\mathbb{R}\\)。我们平时真正关心的不是底层的 \\(\omega\\)，而是 \\(X(\omega)\\) 在实数轴上的取值。

#### 3.3 分布是什么：随机变量诱导出的测度

有了随机变量 \\(X\\) 之后，我们通常关心的问题是：

> \\(X\\) 落在某个集合 \\(A\\) 里的概率是多少？

这里的 \\(A\\) 是取值空间 \\(\mathcal{X}\\) 中的集合。可是原始概率测度 \\(\mathbb{P}\\) 是定义在样本空间 \\(\Omega\\) 上的，不能直接作用在 \\(A\\) 上。因此，我们需要先把 \\(A\\) 拉回到样本空间：

\\[
X^{-1}(A) = \\{ \omega \in \Omega: X(\omega) \in A \\}
\\]

这个 \\(X^{-1}(A)\\) 才是样本空间 \\(\Omega\\) 里的事件，所以可以对它使用 \\(\mathbb{P}\\)。于是我们定义：

\\[
\mu(A) = \mathbb{P}(X^{-1}(A))
\\]

这个 \\(\mu\\) 就是随机变量 \\(X\\) 在取值空间 \\(\mathcal{X}\\) 上诱导出的分布测度。

#### 对几个关系梳理的总结

> What is a distribution? It is a measure induced by a random variable on the value space.

> What is a measure? It is a function that assigns a non-negative number to each measurable set, satisfying certain properties.

> What is a random variable? It is a function that maps outcomes from the sample space to the value space.

#### 这里有一个混淆点,概率测度到底是 \\(\mathbb{P}\\) 还是 \\(\mu\\)？概率空间是定义了两个测度吗?

- 概率空间只定义了一个测度：\\(\mathbb{P}\\)。
- \\(\mu\\)是随机变量 \\(X \\)在取值空间上“推出来的”另一个测度。
- 它们不是同一个测度，但 \\(\mu\\)是由\\(\mathbb{P}\\)诱导出来的。 pushforward measure（推前测度）



严格写法：

\\[
X \sim \mu
\\]

如果 \\(\mu\\) 有密度函数 \\(p(x)\\)，那只是：

\\[
d\mu(x) = p(x)dx
\\]

\\(p(x)\\)密度是测度的“微分形式”。
因此，严格地说，**分布首先是一个测度，我们只是用密度函数表示分布**。

#### 3.4 期望是什么：对测度积分

从原始概率空间 \\(\Omega\\) 上看，期望的本质是:

\\[
\mathbb{E}[f(X)] = \int_{\Omega} f(X(\omega)) d\mathbb{P}(\omega)
\\]

这表示：我们对样本空间中的所有实验结果 \\(\omega\\) 积分，每个 \\(\omega\\) 按照概率测度 \\(\mathbb{P}\\) 加权。

但因为随机变量 \\(X\\) 已经把 \\(\mathbb{P}\\) 推到了取值空间 \\(\mathcal{X}\\) 上，所以同一个期望也可以写成对分布测度 \\(\mu\\) 的积分：

\\[
\mathbb{E}_\mu [f(X)] = \int_{\mathcal{X}} f(x)d\mu(x)
\\]

如果 \\(\mu\\) 有密度 \\(p(x)\\)，即 \\(d\mu(x)=p(x)dx\\)，那么上式就变成我们更熟悉的形式：

\\[
\mathbb{E}[f(X)] = \int_{\mathcal{X}} f(x)p(x)dx
\\]

这就是期望的密度函数写法。这里的 \\(x\\) 是积分变量，也就是 dummy variable；它只是积分表达式里的占位符，不是随机变量本身。

#### 3.5 所以：dummy variable “服从分布”并不是真的在“服从”

你看到的：

\\[
\mathbb{E}_{x\sim p(x)}[f(x)]
\\]

严格展开其实是：

\\[
\int f(x)p(x)dx
\\]

或者从测度论角度：

\\[
\mathbb{E}_\mu[f] = \int fd\mu
\\]

因此：

- 角标里的 “\\(x\sim p(x)\\)” **不是严格的“服从分布”语义**  
- 它只是一个 **非严格的、工程化的 shorthand**，用来提醒读者：  
  “积分变量叫 x，密度是 p(x)”

所以它不会与“分布必须作用在随机变量上”产生真正的数学矛盾——只是符号层面容易让人误解。

---

#### 3.5 深度学习论文中的典型混写

在 DL 论文中，你会看到：

\\[
x \sim p(x), \quad \mathbb{E}_{x\sim p(x)}[f(x)]
\\]

它们通常同时承担多个角色：

- \\(x\\)：既是“随机变量”的名字，又是“取值”的符号  
- “\\(\sim\\)”：既像“服从分布”，又像“从分布采样”  
- 角标：既像“分布”，又像“积分测度”

从严格数学角度看，这当然是混的；  
但从工程角度看，它足够“好用且不至于误解太多”。

###  小节: 可以怎么写得既严谨又不啰嗦？

如果你想在论文中写得 **既严谨又清晰**，可以采用这样的风格：

- \\(X\\) 表示随机变量，用 \\(x\\) 表示取值或者积分变量,用 \\(x_i\\) 表示样本
- 分布：  
  \\[
  X \sim p(x)
  \\]
- 期望：  
  \\[
  \mathbb{E}[f(X)] = \int f(x)\, p(x)\, dx
  \\]
- 也可以写成：  
  \\[
  \mathbb{E}_{x\sim p(x)}[f(x)]
  \\]
  并在符号说明中注明：  
  “\\(\mathbb{E}_{x\sim p(x)}\\) 表示对密度 \\(p(x)\\) 的积分，\(x\\) 为积分变量。”

这样既尊重测度论的本质，又兼顾读者习惯。

本质上是对积分形式的压缩。它好用、直观，但并不是最严格的概率论符号。做理论推导时，最好先用 \\(X\\) 表示随机变量，用 \\(x\\) 表示取值或者积分变量,用 \\(x_i\\) 表示样本，再根据上下文决定是否采用 ML 风格的简写。
